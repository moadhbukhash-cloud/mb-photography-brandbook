# MB Photography — Brand Book

The visual identity system for Moadh Bukhash Photography.

Live: [moadh.com/brandbook](https://moadh.com/brandbook)

## Structure

13 chapters across 4 MECE groups:
- **Expression** — Story, Voice, Photography
- **Form** — Logo, Color, Type, Pattern, Anchor
- **Frame** — Layout, Motion
- **Practice** — Production, Applications

## Build

Static HTML — no build step. Deploy the entire folder to any static host.

## Photography

All hero photographs in `assets/photos/` are watermarked, downsampled to 1600px max edge.
Originals live outside the deploy artifact (protectionism).
